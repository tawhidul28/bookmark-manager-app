# 📑 Bookmark Manager App

A modern and responsive Bookmark Manager built with **HTML, CSS, and JavaScript**. Easily save, organize, search, sort, archive, and manage your favorite websites in one place.


## 🚀 Live Demo

🔗 **Live Website:** https://tawhidul28.github.io/bookmark-manager-app/


## ✨ Features

- ➕ Add new bookmarks
- ✏️ Edit existing bookmarks
- 🗑️ Delete bookmarks
- 📦 Archive & Unarchive bookmarks
- 📌 Pin important bookmarks
- 🔍 Search bookmarks by title
- 🏷️ Organize bookmarks with tags
- 📊 Sort bookmarks
  - Recently Added
  - Recently Visited
  - Most Visited
- 💾 Data stored using Local Storage
- 📱 Responsive Design
- 🎨 Clean and Modern UI
- 🔔 Toast notifications
- 👤 Profile dropdown menu
